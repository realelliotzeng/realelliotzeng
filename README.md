## About Me

I'm Elliot, a front-end engineer who enjoys building polished, practical, and visually thoughtful interfaces.

I currently work on desktop and interactive UI projects, with a strong interest in Tauri-based applications, modern front-end engineering, and product-oriented development. I care about how software feels in use, not only how it works.

- Building with Vue, React, TypeScript, and Tauri
- Learning more about Go and backend fundamentals
- Interested in product design, interaction details, and developer workflows
- Enjoying photography, fitness, music, and philosophy outside of coding

## Current Focus

- Crafting clean UI for desktop tools and internal systems
- Turning ideas into usable products through fast iteration
- Strengthening engineering depth while keeping strong design sensitivity
- Exploring the balance between product thinking and implementation quality
